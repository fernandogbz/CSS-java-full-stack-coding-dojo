JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons. It is a small a lightweight language allowing maximum flexibility for developers to take in a bunch of diferent directions. JavaScript lices inside a host enviroments (a web server browser or Node server), it can be connected to objects of these enviroments to provide programmatic control over them.
